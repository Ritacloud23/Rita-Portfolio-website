VPC-Based cloud Infrastructure Deployment

Designed and deployed a secure and highly available AWS Virtual Private Cloud (VPC) architecture featuring public and private subnets, custom route tables, a NAT Gateway, and security groups. Configured EC2 instances for hosting, implemented IAM roles for controlled access, and integrated CloudWatch for real-time monitoring and performance optimization. This setup ensures network isolation, scalability, and robust security for cloud-based applications.
